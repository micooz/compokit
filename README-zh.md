## 访问地址

https://compokit.apporz.com

## 功能

- 可交互虚拟钢琴键盘。
- 易用的和声进行设计器。
- 包含 24 个大小调、7 个中古调式的音阶与和弦。
- 移动设备可用。
- 离线可用。
